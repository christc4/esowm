# even shittier opinionated window manager

## about

esowm strips [sowm](//github.com/dylanaraps/sowm>sowm) of vanity

## <s>Features</s>

- No workspaces
- No Alt-Tab
- No fullscreen

Read [Why Tiling Window Managers suck, X.Lee](http://xahlee.info/linux/why_tiling_window_manager_sucks.html)

## Commands

- Super + q, kill window
- Super + d, dmenu
- Super + Enter, Spawn terminal
- Super + Mouse Button 1, manual window raise
